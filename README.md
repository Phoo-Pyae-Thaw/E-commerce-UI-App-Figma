🍔 Food Delivery App UI (Figma)

-Overview
This project is a simple food delivery mobile app UI designed in Figma.
It includes user authentication, category browsing, restaurant selection, and order flow.

Features
* Home screen introduction
* Login & Register system
* Food categories (3 categories)
* Restaurant selection (3 per category)
* Menu selection (3 items per restaurant)
* Order summary
* Order success confirmation

-Design

* Clean and minimal UI
* Pink theme inspired by modern delivery apps
* Built using Auto Layout in Figma
